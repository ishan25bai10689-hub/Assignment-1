# Student Report Card System (Assignment-1)

Hi! Thanks for checking out my project.

This is a straightforward repository where I am storing code for a **Student Report Card System**. This is a Python script that takes student details and marks, calculates their grades, and saves the records for future reference.

I built this because I wanted to practice **Python basics (Input handling, conditional logic, and File I/O)** and see how to manage student data in a simple way.

##  What this project does

* **Automated Grading:** Takes marks for 3 subjects and calculates the average and final grade (A, B, C, D, or Fail).
* **Data Persistence:** Automatically appends every student's record (Name, Reg No, Grade) to a text file named `class_records.txt`.
* **Error Handling:** Prevents the program from crashing if a user enters invalid text instead of numbers.
* **Continuous Loop:** Allows the user to enter multiple students one after another without restarting the program.
